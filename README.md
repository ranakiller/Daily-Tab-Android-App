**DailyTab — Daily Purchase Tracker**

DailyTab is a mobile-first progressive web app designed to help you track daily purchases from your regular shops and calculate exactly what you owe at the end of each month — no more mental math, no more disputes with your shopkeeper.

**The problem it solves**

If you buy milk, bread, or other daily essentials on credit from a local shop and settle the bill monthly, keeping track manually is a pain. DailyTab handles it automatically.

**Key Features**

- **Multi-shop support** — Add 2–3 regular shops, each color-coded for quick identification
- **Item catalog** — Save your regular items with default prices per shop; prices auto-fill when logging
- **Daily logging** — Log purchases in seconds by picking a shop, selecting an item, and setting quantity
- **Calendar view** — Visual monthly calendar with colored dots showing which days you bought from which shop; tap any date to add or review entries
- **Purchase frequency tracker** — See exactly how many days you bought each item (e.g. Milk: 24 days ✓, 7 days missed ✗) with a visual progress bar
- **Monthly summary** — Per-shop breakdown of total spent, amount paid, and amount still owed with a day-by-day drill-down
- **Payment tracking** — Log partial or full payments against each shop; track remaining balance
- **Theme support** — Light, Dark, and System (follows device) themes
- **Custom currency** — Set any currency symbol; saved symbols stay as quick-select options
- **Offline-ready** — Data persists via localStorage, making it fully functional as an installed APK

**Tech Stack**

React 18 · Inline styles · localStorage + Claude artifact storage · No external UI libraries · PWA-ready

**Installation**

Use as a web app directly, or wrap with [WebIntoApp](https://webintoapp.com) or [PWABuilder](https://pwabuilder.com) to generate an installable Android APK in minutes.
